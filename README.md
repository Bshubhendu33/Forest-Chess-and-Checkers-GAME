## 🎮 What is this?

A fully-featured **Chess & Checkers** game set in a living forest world — no installs, no dependencies, no build step. Just open `index.html` and play.

Two kingdoms clash: the **Light Kingdom** (Lion, Eagle) vs the **Shadow Kingdom** (Tiger, Owl), with shared pieces — Elephants, Foxes, Horses, and Boars — serving both sides.

## ✨ Features

- ♟️ **Full Chess Rules** — en passant, castling, check, checkmate, stalemate, promotion
- 🎲 **Checkers Mode** — king promotion, multi-jump capture chains
- 🎭 **Animated Pieces** — each animal has its own idle animation (roaring lions, soaring eagles, stomping rhinos)
- 🔊 **Procedural Animal Sounds** — every piece has a unique voice, zero audio files
- 🌐 **Multiplayer** — create a room, share a 6-char code, opponent joins instantly
- ⏱️ **Chess Clock** — 10 min per player with low-time warning
- ↩️ **Unlimited Undo** — full board state serialized every move
- 📜 **Move Log** — algebraic notation, live and scrollable
- 💡 **Move Hints** — legal moves highlighted on selection
- 🌲 **Forest Theme** — fireflies, falling leaves, 3D board with wood-grain tiles
- 📱 **Responsive** — desktop, tablet, and mobile
